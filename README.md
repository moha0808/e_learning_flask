# E-Learning Platform (Flask)

A comprehensive web-based e-learning application built with the Flask framework. This platform allows instructors to manage courses and students to access educational content seamlessly.

## 🚀 Features

* **User Authentication:** Secure login and registration for Students and Instructors.
* **Course Management:** Instructors can create, edit, and delete courses.
* **Interactive Learning:** Students can enroll in courses and track their progress.
* **Responsive Design:** Optimized for both desktop and mobile viewing.

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3, JavaScript (Jinja2 templates)
* **Database:** SQLite (or your preferred DB)
* **Styling:** Bootstrap 5

## 📦 Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/moha0808/e_learning_flask.git](https://github.com/moha0808/e_learning_flask.git)
    cd e_learning_flask
    ```

2.  **Create a virtual environment:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows use: venv\Scripts\activate
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Run the application:**
    ```bash
    python app.py
    ```
    The app will be available at `http://127.0.0.1:5000`.

## 📂 Project Structure

* `app.py`: The main entry point of the application.
* `templates/`: Contains HTML files.
* `static/`: Contains CSS, images, and JavaScript files.
